# InvoiceApp Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Повний редизайн InvoiceApp — Design System First, Teal/Mint палітра, Inter шрифт, Wide Sidebar layout з mobile Drawer, Dashboard stats page.

**Architecture:** Спершу встановлюємо font + Tailwind токени, потім переписуємо `AuthenticatedLayout.vue` (sidebar + drawer), оновлюємо shared components, додаємо `DashboardController` зі статистикою, і застосовуємо нові стилі до кожної сторінки.

**Tech Stack:** Vue 3, Inertia.js, Tailwind CSS v3, Laravel 11, PHP 8.2

```php
<?php
// app/Http/Controllers/DashboardController.php

namespace App\Http\Controllers;

use App\Models\Invoice;
use Inertia\Inertia;

class DashboardController extends Controller
{
    public function __invoke()
    {
        $stats = [
            'total_amount'   => Invoice::sum('total'),
            'paid_amount'    => Invoice::where('status', 'paid')->sum('total'),
            'pending_amount' => Invoice::where('status', 'sent')->sum('total'),
            'overdue_amount' => Invoice::where('status', 'overdue')->sum('total'),
            'total_count'    => Invoice::count(),
            'paid_count'     => Invoice::where('status', 'paid')->count(),
            'pending_count'  => Invoice::where('status', 'sent')->count(),
            'overdue_count'  => Invoice::where('status', 'overdue')->count(),
        ];

        return Inertia::render('Dashboard', compact('stats'));
    }
}
```
